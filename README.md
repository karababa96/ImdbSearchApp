# ImdbSearchApp MVVM

##### In this app, I use omdb API in order to reach some information abaout film and tv series such as; 
* Film Name,
* Actors,
* Writer,
* Director,
* Year,
* Runtime,
* Language,
* Awards,
* Pilot.
##### While I was creating the user interface, I used SwiftUI Components.
* ListView
* NavigationView
* Text
#### In order to show poster of Films and Tv Series correctly, I have created special Image view in different SwiftUI file. Then, I have adjusted some option to obtain beauty posters.
